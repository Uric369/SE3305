# 作业2：2D Game with Physics

## 文件说明
- `Assets`：包含了游戏的资源文件，比如图片、动画等。
- `ProjectSettings`：包含了Unity项目的设置文件。
- `Packages`：包含了本项目设计的所有Unity插件、依赖。

## 提交内容
- `handin`文件夹包括：
    - `build`：游戏构建结果，可以直接运行的游戏文件。
    - `package`：资源包和依赖项。
- `hw2 2D-Game-with-Physics 游戏说明.pdf`
